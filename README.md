# Kenji Kagawa Sitemap

This repository hosts the sitemap for the [Kenji Kagawa website](https://tinyurl.com/kkagawa). The sitemap helps search engines like Google discover and index all pages of the site efficiently.

## Sitemap File

- `sitemap.xml` — Contains all public pages of the website, including:
  - Home
  - About
  - Booking / Contact
  - FAQ
  - Gallery
  - Legal pages (Terms of Service, Privacy Policy, Client Consent & Waivers)

## How to Use

1. Open the sitemap URL in your browser to verify it is accessible.  
   Example: `https://yourusername.github.io/kkagawa-sitemap/sitemap.xml`

2. Submit the sitemap URL to [Google Search Console](https://search.google.com/search-console/) or other search engines to improve indexing.

3. Ensure your website pages are public and linked from the homepage to maximize discoverability.

## Notes

- Keep this repository public so search engines can access the sitemap.  
- Update the `sitemap.xml` whenever you add, remove, or rename pages on your website.  
- Google Sites automatically generates a virtual sitemap, but this custom sitemap provides explicit guidance for search engines to index key pages and contact information.
